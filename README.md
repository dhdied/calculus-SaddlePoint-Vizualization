# Визуализация седловой точки (UltimateSaddlePoint)

Проект на базе библиотеки Manim для визуализации математической функции $z = (y - x^2)(y - 3x^2)$. Проект наглядно демонстрирует случай, когда классическое достаточное условие (через определитель матрицы Гессе) не дает ответа в критической точке $(0,0)$, хотя на самом деле там находится седловая точка.

## Описание
Анимация пошагово разбирает:
1.  **Необходимое условие**: расчет градиента и поиск критической точки $M(0,0)$.
2.  **Достаточное условие**: расчет вторых производных и построение матрицы Гессе, определитель которой равен 0.
3.  **3D-визуализацию**: построение поверхности и демонстрацию "ловушки" — по всем прямым направлениям точка кажется минимумом, но в узкой зоне между двумя параболами функция уходит вниз.

## Требования

Для успешного запуска проекта в системе должны быть установлены:

* **Python 3.8+**
* **Manim (Community Edition)**
* **FFmpeg** (необходим для кодирования видео)
* **LaTeX** (например, MiKTeX или TeX Live) — требуется для рендеринга математических формул.
* **dvisvgm** (обычно входит в состав дистрибутивов LaTeX)

## Установка и настройка

1.  **Клонирование репозитория**:
    ```bash
    git clone [https://github.com/dhdied/SaddlePoint-Visualization.git](https://github.com/dhdied/SaddlePoint-Visualization.git)
    cd SaddlePoint-Visualization
    ```

2.  **Создание виртуального окружения** (рекомендуется):
    ```bash
    python -m venv manim_env
    manim_env\\Scripts\\activate
    ```

3.  **Установка Manim**:
    ```bash
    pip install manim
    ```
